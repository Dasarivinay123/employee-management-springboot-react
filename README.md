
# Employee Management System

Employee Management System built using Spring Boot, React, MySQL, Docker, Jenkins and AWS EC2.

## Live Application

Frontend: http://43.204.68.152  
Backend API: http://43.204.68.152:8080

## Features

- User Registration & Login
- JWT Authentication
- Role Based Access (Admin / User)
- Email Verification
- Forgot Password / Reset Password
- Add, Update, Delete Employees
- Search & Pagination
- Responsive UI

## Tech Stack

- Java
- Spring Boot
- Spring Security
- React JS
- MySQL
- Docker
- Jenkins
- AWS EC2

## CI/CD

- Code pushed to GitHub
- Jenkins build triggered
- Docker image created
- Deployed to AWS EC2
